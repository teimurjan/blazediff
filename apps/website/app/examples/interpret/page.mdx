Structured image diff analysis powered by a native Rust binary. Unlike pixel-level or perceptual comparison methods, `interpret` identifies **change regions** and classifies them semantically — telling you _what_ changed and _where_.

## Installation

```bash npm2yarn
npm install @blazediff/interpret-native
```

## Overview

The interpret function analyzes two images and returns structured results including severity, diff percentage, and detailed region analysis with content-aware change types (Addition, Deletion, Shift, ContentChange, ColorChange, RenderingNoise).

import { interpret } from "@blazediff/interpret-native";

const result = await interpret("fixtures/3a.png", "fixtures/3b.png");
console.log(result.summary);
console.log(`Severity: ${result.severity}`);
console.log(`Changed: ${result.diffPercentage.toFixed(2)}%`);

for (const region of result.regions) {

### Compact Mode

Use compact mode to get a simplified result with only the essential information:

```ts
import { interpret } from "@blazediff/interpret-native";

const result = await interpret("fixtures/3a.png", "fixtures/3b.png", {
  compact: true,
});
// Returns: { summary, severity, diffPercentage, regions: CompactRegion[] }
```
