# Pixel-by-pixel Comparison - Vanilla JavaScript

import { Tabs } from "nextra/components";
import DiffImageComparison from "../../../../components/diff-image-comparison";

Compare two images pixel by pixel in pure JavaScript - no native binaries, no
WebAssembly. `@blazediff/core` runs anywhere JS runs (browser, Node, Deno, Bun,
edge) and is ~1.5× faster than pixelmatch. Reach for it when you want zero install
friction and portability over raw throughput. Need more speed? See
[🐆🐆 Rust + WASM](/docs/pixel-comparison/rust-wasm) and
[🐆🐆🐆 Rust + N-API](/docs/pixel-comparison/rust-napi).

## Installation

```bash npm2yarn
npm install @blazediff/core
```
