Image and object diffing. Rust native binary with SIMD, pure JS core, perceptual metrics (SSIM, GMSD), structured interpret mode, and test framework matchers for Jest/Vitest/Bun.

## Quickstart

```sh npm2yarn
npm i @blazediff/core-native
```
